But is it safe?

The code contained in this bookmarklet is hosted completely locally, meaning that the secret code you enter never leaves your browser. When you enter it into the generator, it is processed only on your own pc to create the tool. The tool works by processing the secret code, together with the current time, to generate the six digits OTP that you have to enter in the UU services, in the same way as your Google Authenticator app would. This is possible due to the fact that the algorithm that is used to get this OTP is publicly available. The only risk that you have to take in mind is that 2-Factor Authentication is meant as an extra measure of security for your account, since it forces you to use a separate device (i.e. your phone) to verify that it is actually you trying to log in.
